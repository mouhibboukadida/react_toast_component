## React Toast Notification Component

A custom toast notification system built from scratch using React.
No external libraries were used.

### Features
- Multiple toast types (Success, Danger, Info, Warning)
- Auto dismiss after delay
- Manual close button
- Fixed position (bottom-right)
- Simple animation
- Reusable component

### Tech Stack
- React
- JavaScript
- CSS

### Purpose
This project was built to better understand:
- React hooks (useState, useEffect, useCallback)
- Component communication (props)
- UI logic without third-party libraries
